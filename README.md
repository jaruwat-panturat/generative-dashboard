# Generative Dashboard

A dashboard that displays a static UI by default and generates dynamic UIs based on user prompts.

## Features
- Static default dashboard on first load
- AI-powered widget generation from natural language prompts
- Persistent dashboard layouts and configurations

## Getting Started

### Prerequisites
- Node.js 20+
- Anthropic API key

### Installation
```bash
npm install
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env
```

### Development
```bash
npm run dev
```
